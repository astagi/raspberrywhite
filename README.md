Raspberrywhite
==============
Your tiny social player. Use your MP3 collection to create a modern, portable and funny jukebox with Raspberry Pi.

Install dependencies
--------------------

    $ pip install -r requirements.txt

Make it running
---------------

    $ ./run.sh
